Data Model-Based Ingestion Pipeline (dm-bip)
==========================================================================

dm-bip is a data ingestion pipeline that uses `LinkML <https://linkml.io/linkml/>`_ tools to transform tabular scientific data into a harmonized data model. It infers schemas from source data, validates against those schemas, and transforms data to a target model using declarative mapping specifications.

Currently used by the `BioData Catalyst <https://biodatacatalyst.nhlbi.nih.gov/>`_ Data Management Core and the `INCLUDE <https://includedcc.org/>`_ project.
